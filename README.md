# Backend

## Tecnologias

- [Node.js](https://nodejs.org/)
- [Nest](https://nestjs.com/)
- [Typeorm](https://typeorm.io/#/)
- [Typescript](https://www.typescriptlang.org/)
- [MySQL](https://www.mysql.com/)

## Como executar

- Instale o `docker`
- Clone o repositório
- Execute nesse diretório `docker-compose up`
